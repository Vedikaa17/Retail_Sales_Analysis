# Retail Sales Analysis Dashboard

## Objective
The objective of this project is to analyze retail sales data to identify sales trends, customer behavior, product performance, and regional revenue patterns. An interactive Power BI dashboard is created to visualize key insights.

## Dataset
- Dataset Name: Superstore Sales Dataset
- Source: Kaggle
- File Used: superstore_cleaned.csv
- Type: Retail transactional data

## Power BI Dashboard Features
- KPI cards showing:
  - Total Customers
  - Total Sales
  - Total Orders
- Sales by Category (Technology, Office Supplies, Furniture)
- Sales by Region (Central, East, South, West)
- Top 10 Customers by Sales
- Monthly Sales Trend (January to December)
- Interactive Region slicer to filter all visuals dynamically

## Key Insights
- Sales show clear monthly and seasonal trends
- Technology category contributes the highest revenue
- A small set of customers generate a large portion of sales
- Certain products consistently underperform
- Regional analysis highlights strong and weak markets
